## Description

This is a messenger prototype built with **React** in **ES6**. In order to demostrate its practicality, I integrated the chatbot logic I previously created ([repo](https://github.com/embrilliant/talking-bot)).

## Technology

* React
* Build tool: Webpack

## Development Setup

```
npm install
```

### Build Project

```
npm run build
```
